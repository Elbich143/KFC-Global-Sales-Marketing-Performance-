# KFC-Global-Sales-Marketing-Performance-
# Project Overview:-
This project analyzes global sales and marketing performance of KFC outlets using SQL and Power BI. The goal is to extract meaningful business insights, evaluate marketing effectiveness, and identify high-performing regions and branches.

# Objectives:- 
Analyze sales performance across different countries and branches
Evaluate marketing efficiency using ROI (Return on Investment)
Identify trends in monthly sales performance
Understand customer contribution to revenue
Support data-driven decision-making

# Dataset:- 
The dataset contains historical sales data with the following fields:

Year
Month
Country
Branch_ID
Sales
Customers
Marketing_Spend

# SQL Analysis:-
Performed data analysis using SQL to answer key business questions:

# Key SQL Operations:
Aggregations (SUM, COUNT, AVG)
Window Functions (RANK, LAG)
Subqueries
Case-based segmentation

# Key Insights:
Identified top-performing countries by total sales
Found highest revenue-generating branches
Calculated marketing ROI across regions
Analyzed year-over-year sales growth
Detected seasonal trends in monthly sales
